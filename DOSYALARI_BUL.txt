ANDROID STUDIO'DA DOSYALARI BULMA REHBERİ
=========================================

Dosyalar FİZİKSEL OLARAK VAR! Terminal'de kontrol ettik.

Android Studio'da görmek için:

1. File → Sync Project with Gradle Files (EN ÖNEMLİ!)
2. File → Invalidate Caches / Restart
3. Android Studio'yu kapat-aç
